# Structural Triangulation Method (STM)

**A method for reconstructing stable system structures through independent AI models**

---

## 🔍 Overview

The **Structural Triangulation Method (STM)** is an approach for analyzing complex systems by aggregating multiple independent AI model outputs and extracting their shared structural patterns.

Instead of selecting a single “best” answer, STM identifies:

* consistent factors
* recurring causal relationships
* stable system mechanisms

---

## 🧠 Core Principle

> Truth emerges from structural consistency across independent interpretations.

---

## ⚙️ Method Summary

STM consists of the following steps:

1. **Independent Analysis**
   Multiple AI models analyze the same system using identical input

2. **Hard Filtering**
   Remove only structurally invalid or logically inconsistent outputs

3. **Model Evaluation**
   Classify outputs (STRONG / GOOD / WEAK / INVALID)

4. **Factor Aggregation**
   Identify stable value clusters across models

5. **Mechanism Extraction**
   Extract recurring causal steps

6. **Mechanism Reconstruction**
   Build a unified causal chain

7. **Consistency Check**
   Ensure structural coherence

---

## 🧩 Output

The result is:

* a **structural model**
* a **causal mechanism**
* a **reproducible representation of the system**

---

## 🌍 Applications

STM can be applied to:

* legal and regulatory systems
* economic mechanisms
* institutional analysis
* governance structures

---

## 📊 Use Cases (Current)

The method has been tested on:

* EU State Aid regulatory system
* German lobbying and procurement structures
* US regulations (CTA, FARA)

Across multiple independent runs:

* high factor convergence
* consistent mechanisms
* reproducible results

---

## 🔓 Accessibility

STM is designed to be:

* usable with publicly available AI models
* reproducible
* transparent

It lowers the barrier to structured analysis of complex systems.

---

## ⚠️ Limitations

STM does not provide:

* quantitative predictions
* statistical estimates

It provides:

* structural relationships
* causal mechanisms
* directional insights
